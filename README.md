## Description

A TDD-based implementation of the String Calculator Kata in Dart.

## Features

- Supports unlimited numbers
- Comma & newline delimiters
- Custom delimiters
- Multiple custom delimiters
- Throws exception for negative numbers
- Lists all negative numbers in exception
